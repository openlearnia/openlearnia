# PDF Tools MVP Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship a privacy-first Astro PDF suite (hub + merge + split + rotate) at `projects/tools/pdf-tools`, client-side only, listed on openlearnia.com/tools.

**Architecture:** Mirror `projects/tools/image-tools`: static Astro site, dark tokens, DropZone/ToolShell chrome, `pdf-lib` mutations in a Web Worker via an engine queue, `pdfjs-dist` thumbnails on the main thread (canvas needs DOM — do not force thumbs into the worker).

**Tech Stack:** Astro 5, TypeScript, `pdf-lib`, `pdfjs-dist`, `@astrojs/sitemap`, Bun, Cloudflare Pages (`wrangler.toml` name `pdf-tools`).

## Global Constraints

- Privacy-first: files never leave the device; no upload APIs; privacy badge + footer copy on every page.
- Limits: `MAX_FILE_BYTES = 50 * 1024 * 1024`, `MAX_PAGES = 200`.
- Password PDFs: hard-fail with exact message: `This PDF is password-protected. Unlock support isn't available yet — use an unlocked copy.`
- Corrupt PDFs: `Couldn't read this PDF. It may be damaged.`
- MVP tools only: Merge, Split, Rotate — no compress/OCR/Office.
- Do not git commit unless the user explicitly asks (this session: leave uncommitted).
- Scaffold in-tree at `projects/tools/pdf-tools`. Human follow-up: `gh repo create`, submodule entry, Pages project, DNS for `pdf-tools.openlearnia.com`.

```ts
// ranges.ts — 1-based inclusive → 0-based [start, end] inclusive indices
export function parsePageRange(from: number, to: number, pageCount: number): { start: number; end: number } | string {
  if (!Number.isInteger(from) || !Number.isInteger(to)) return 'Page numbers must be whole numbers.';
  if (from < 1 || to < 1 || from > pageCount || to > pageCount) return `Pages must be between 1 and ${pageCount}.`;
  if (from > to) return 'Start page must be ≤ end page.';
  return { start: from - 1, end: to - 1 };
}
```
